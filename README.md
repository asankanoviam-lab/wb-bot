# WB Авто-ответы на отзывы (Gemini)

## Установка на Railway (бесплатно)

### Шаг 1 — Получить ключи
- WB API: личный кабинет WB → Настройки → Доступ к API
- Gemini API: aistudio.google.com → Get API Key → Create API Key (бесплатно)

### Шаг 2 — Загрузить на GitHub
1. Зайдите на github.com → New repository → создайте репо
2. Загрузите все файлы из этой папки

### Шаг 3 — Запустить на Railway
1. Зайдите на railway.app → New Project
2. Deploy from GitHub repo → выберите ваш репозиторий
3. Variables → добавьте переменные:
   - WB_API_KEY = ваш ключ WB
   - GEMINI_API_KEY = ваш ключ Gemini
   - CHECK_INTERVAL_MINUTES = 30

### Что делает бот
- Каждые 30 минут проверяет новые отзывы без ответа
- Gemini генерирует ответ с учётом оценки:
  - 1-2 звезды → извиняется, предлагает решение
  - 3 звезды → благодарит, учитывает замечания
  - 4-5 звезд → тепло благодарит, приглашает снова
- Автоматически отправляет ответ на WB
